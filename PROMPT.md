# Project: Redesign oracleapex.cn

**Goal:** Create 3 distinct UI/UX design concepts for oracleapex.cn using ui-ux-pro-max skill.
**Reference Style:** Oracle Official Website (Clean, Enterprise, Professional, Red/Black/Grey/White palette).

## Current Site Content
- APEX 问题求助 (Forum/Help)
- APEX 新手入门 (Beginner Guide)
- APEX 进阶技巧 (Advanced Tips)
- 建议与意见反馈 (Feedback)

## Task
1. Analyze the requirements.
2. Use `ui-ux-pro-max` logic to generate 3 design systems.
3. For each design (Concept A, B, C), provide:
   - **Style Name** & **Vibe**
   - **Color Palette** (Hex codes)
   - **Typography** choices
   - **Layout Structure** (Hero, Grid, Navigation)
   - **Key Component Styling** (Buttons, Cards)
4. Create a `DESIGNS.md` file with these details.
5. Create simple HTML/CSS mockups for the "Hero Section" of each concept in `concept-a.html`, `concept-b.html`, `concept-c.html`.

**Constraint:** Do NOT try to login to the live site. Just design based on the public structure and Oracle style guide.
